🎯 Our Mission

This research demonstrates a modular, multi-agent framework that autonomously navigates the early-stage drug discovery pipeline. By synergizing large language models with robust machine learning classifiers, we've created a transparent blueprint for an integrated discovery engine—aligned with OSDD2's vision of democratizing pharmaceutical innovation.

Autonomous Discovery Framework

A coordinated system of specialized AI agents working in harmony to execute complex drug discovery workflows

End-to-End Pipeline

1

Target Identification

Abstract Mining Agent queries PubMed using complex Boolean logic to discover novel therapeutic targets for Alzheimer's Disease

2

Target Validation

Target Evaluator Agent scores each candidate on novelty, evidence strength, and confidence using web search tools

3

Model Construction

Planner & Coding Agents build XGBoost classifiers for bioactivity and ADME/Tox prediction using ChEMBL and TDC datasets

4

Molecule Generation

Molecule Generator (NVIDIA MolMIM) creates 30 novel compounds per cycle from validated seed inhibitors

5

Multi-Property Filtering

Molecule Evaluator applies threshold-based cascade: hERG toxicity → solubility → stability → potency

6

Iterative Refinement

Top hits become seeds for next generation cycle, continuously optimizing toward multi-property objectives

🎯

Specialized Agents

Abstract Mining, Target Evaluator, Planner, Coder, Medicinal Chemist, Molecule Generator, and Evaluator agents—each expert in their domain

🔄

Iterative Design

Closed-loop system where each cycle refines molecular candidates based on multi-objective optimization criteria

📊

Transparent Pipeline

Every decision is traceable—from literature mining to final molecule selection—enabling reproducible research

🎯 Key Achievements

  • Generated structurally novel scaffolds distinct from seed molecules
  • Achieved >0.90 predicted bioactivity for top candidates
  • Successfully filtered for favorable ADME/Tox properties
  • Demonstrated scaffold hopping capability

⚠️ Critical Insights

  • Data quality fundamentally limits AI performance
  • CGAS model failure highlights data scarcity challenges
  • Manual threshold curation improved class balance
  • Human-in-the-loop oversight remains essential

💡 Key Finding

The framework's failure to build a reliable model for CGAS (only 246 data points) highlights a crucial limitation: autonomous AI systems are fundamentally tethered to data availability and quality. This reinforces the need for human expert oversight in data curation and model validation—a core principle of OSDD2's collaborative approach.
